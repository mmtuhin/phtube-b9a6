Discuss the scope of var, let, and const

If var is declared outside of a function, its scope is global. var declared inside of a function is called local scope. var can be redeclared and updated.

Let and const is new compared to var. These are block scope variable. Variable declared with let can be updated but can't be re-declared in its scope.

const is used to store constant values which will never be changed. It's also a local scope variable.

Tell us the use cases of null and undefined

Undefined means the variable has been declared, but its value has not been assigned. Null means an empty value or a blank value.
